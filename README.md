## Setup

```sh
git clone https://github.com/GPT-Engineer-App/app-shell-nav-sidebar-toolbar.git
cd app-shell-nav-sidebar-toolbar
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
